# E-Library Management System (AWS Serverless)

## Description
Serverless backend application built using AWS Lambda, API Gateway, and DynamoDB to manage library operations.

## Features
- Add books
- Issue and return books
- Serverless architecture

## Architecture
- API Gateway → Lambda → DynamoDB
- CloudWatch for logging

## API Endpoints
- POST /addBook
- POST /issueBook
- POST /returnBook

## Tools
- AWS Lambda, API Gateway, DynamoDB, Python, Postman, GitHub
